# apps/admin

Internal Umoja Airways admin dashboard. Staff-only.

**Stack:** Vite + React + TypeScript + MUI + Radix.

**Migrated from:** `umojaairways/booking/uis` on 2026-05-20.

## Dev

```bash
cp .env.example .env.local
npm install
npm run dev                  # Vite — http://localhost:5173
```

## Build

```bash
npm run build
```

Static output in `dist/`.
